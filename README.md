# about-me-website



A responsive personal portfolio website built with **HTML** and **CSS**.  
Showcases projects, services, about section, and contact info with a clean design.  

## Features
- Hero section with full-screen background and readable text  
- Responsive navbar and layout  
- Projects with hover effects  
- About and Contact sections  

## Tech
- HTML5  
- CSS3  
- Google Fonts (Montserrat)
